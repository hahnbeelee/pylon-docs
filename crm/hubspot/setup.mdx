<Note>You'll need a Hubspot administrator to complete the setup.</Note>

1. Visit the [Integrations Page](https://app.usepylon.com/integrations) and hit
   `Connect` under Hubspot

<Frame>
  <img src="/images/hubspot-install.png" />
</Frame>
3. Click `Authorize` on the Hubspot Authorization Page

4. Click `Manage` on the Hubspot Integration and confirm the User Mapping is set
   correctly. The User Mapping allows us map Hubspot Users to Slack Users

<Frame>
  <img src="/images/hubspot-user-mapping.png" />
</Frame>

All done!

<Check>You've setup our Hubspot Integration</Check>
