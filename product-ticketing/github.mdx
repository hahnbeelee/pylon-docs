<Tip>
  The Github Issues integration is likely most useful to you if you're an open
  source company.
</Tip>

## Setup

1. Visit the [Integrations Page](https://app.usepylon.com/integrations) and hit
   `Connect` under Github
2. Authorize the Github App

<Check>You've setup the Github Integration</Check>
